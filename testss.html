QA and Testing for Banking Website and Mobile Applications
(Banki Mobile & Banki Wallet)
for Credit Agricole Egypt

Project Overview:
In this project, I was testing the banking website and two mobile apps—Banki Mobile and Banki Wallet. The goal was to make sure the website and apps were secure, easy to use, and worked well.

Key Contributions:
1. Performed manual testing on the website and both mobile apps to ensure they worked properly.
2. Tested the security of the website and apps to find any potential vulnerabilities.
3. Developed and ran test cases to check the features of the website and apps.
4. Participated in Agile sprints to give feedback and make improvements.
5. Found bugs during testing and reported them with clear steps.
6. Evaluated the UI/UX to ensure the website and apps provided a smooth and user-friendly experience.

E-commerce Website Testing

Project Overview:
In this project, I was testing an entire e-commerce website to make sure it worked smoothly and provided a good experience for users. The goal was to test the website to ensure it provided an easy shopping experience.

Key Contributions:
1. Performed manual testing to check the website’s functionality and usability.
2. Developed test plans and test cases to ensure the website worked as expected.
3. Found bugs during testing and reported them with clear steps to reproduce.
4. Evaluated the UI/UX to ensure the website was user-friendly and intuitive.
